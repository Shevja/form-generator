# Генератор форм

Проект состоит из следующих частей:
1. Поля формы (input, select, textarea, checkbox) и кнопки *src/ui*
2. Генератор форм *src/components/FormGenerator/FormGenerator.vue*
3. 3 демонстрационные страницы

### Генератор форм
Генератор форм принимает на вход схему для вывода полей. Массив объектов слеудующего вида:
```
[
    {
        id: <string>
        type: 'input' | 'select' | 'checkbox' | 'textarea'
        label: <string>
        isValid: <boolean>
        errorMessage: <string>,
        htmlAttrs: HTMLAttributes & Record<string, unknown>,
        
        // Только для type: 'select' 
        options?: [ 
            { 
                label: <string>, 
                value: <string | number> 
            }, 
            ... 
        ]
    },
    ...
]
```
Каждое поле можно кастомизировать через слоты используя нейминг по полю **id**. Также через слот есть доступ к данным поля из схемы через **field** и к привязанной модели через **model**.
Например, если переданная схема включает в себя следующий id:
```js
[
    {
        id: 'username',
        ...
    }
]
```
То можно обратиться к его слоту следующим образом:
```vue
<template #username="{ field, model }"></template>
```


### Демонстрационные страницы:
1. Страница с пустыми формами;
2. Страница с заполненными данными;
3. Страница с кастомными полями;

На каждой странице предоставлена обычная форма и форма с включенными ошибками. 
На странице с заполненными формами, данные идут из **Vuex** эмулируя работу сервера, на этой же странице при нажатии на кнопку "Отправить", данные сохраняются в стор и "отправляются" на сервер. Также происходит сохранение кэша стейта в **localStorage**.
